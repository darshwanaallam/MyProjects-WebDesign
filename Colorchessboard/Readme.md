# Interactive Chessboard Using HTML & CSS

## Overview

This project is an interactive Chessboard built using HTML and CSS. The chessboard consists of an 8×8 grid with alternating black and white squares. When the user hovers over the black squares, they change to different colors, creating a simple interactive effect.

## Features

* 8×8 chessboard layout
* Alternating black and white squares
* Hover color effects on black squares
* Built entirely with HTML and CSS
* Beginner-friendly project
* No JavaScript required

## Technologies Used

* HTML5
* CSS3

## How It Works

### HTML

* Uses a table to create an 8×8 grid.
* Each table cell (`<td>`) represents a square on the chessboard.
* CSS classes are assigned to black squares.

### CSS

* Creates the chessboard pattern.
* Centers the board on the page.
* Adds borders to the board and cells.
* Applies different hover colors to black squares.

## Hover Effects

| Class | Default Color | Hover Color     |
| ----- | ------------- | --------------- |
| w     | Black         | Pale Violet Red |
| x     | Black         | Red             |
| y     | Black         | Gold            |
| z     | Black         | Green           |

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Move your mouse over the black squares to see the color-changing effects.
